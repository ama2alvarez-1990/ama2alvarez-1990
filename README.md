### What I'm building

**[ARIS](https://github.com/ama2alvarez-1990/aris)** is an AI Operating System that unifies Claude, GPT, Gemini, Grok, and Ollama into a single orchestrated intelligence with:

- **19 MCP Tools** — Router, Guardian, Memory, Forge, Scout, Budget
- **53+ Skills** — Code review, security scanning, market research, financial modeling
- **Infinite Memory** — Context persists across sessions and AI models
- **Auto-Evolution** — System discovers and integrates new tools every 24h
- **Privacy-First** — Sensitive data stays on your local models automatically
